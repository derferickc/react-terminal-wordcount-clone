# React Terminal Wordcount Clone

> A simple app that counts the lines, words, and bytes submitted within a form's textarea

> Technology used: React, HTML5, ES6

## Installation / Run Commands

- npm install - to install all packages and dependencies
- npm start - will run the application by default on port 3000 (localhost:3000)
- npm run build - to build the application for production

### Clone

- Clone this repo to your local machine using `git clone https://github.com/derferickc/react-terminal-wordcount-clone.git`
